# 🐱 เปาเบียร์ — แอปบันทึกรายรับรายจ่าย

แอปบันทึกรายรับ-รายจ่ายส่วนตัว สีสันสดใส มีแมวพุงพลุ้ย "เปาเบียร์" เป็นมาสคอตคอยให้กำลังใจ
สร้างด้วย React + Vite พร้อมกราฟสรุปและหมวดหมู่ ใช้งานบนเบราว์เซอร์ได้ทันที ไม่ต้องมีเซิร์ฟเวอร์หรือฐานข้อมูล

## ฟีเจอร์

- บันทึกรายรับ / รายจ่าย พร้อมหมวดหมู่ จำนวนเงิน วันที่ และรายละเอียด
- ดูรายการทั้งหมด กรองตามเดือน ลบรายการได้
- กราฟวงกลมสัดส่วนรายจ่ายตามหมวดหมู่ และกราฟแถบเปรียบเทียบรายรับ-รายจ่าย 6 เดือนล่าสุด
- มาสคอตแมว "เปาเบียร์" เปลี่ยนสีหน้าตามสถานะการเงิน (ยิ้ม / กังวล)
- ข้อมูลเก็บไว้ในเบราว์เซอร์ของผู้ใช้ (localStorage) เป็นส่วนตัวเฉพาะอุปกรณ์นั้น

## รันบนเครื่องตัวเอง

ต้องมี [Node.js](https://nodejs.org) เวอร์ชัน 18 ขึ้นไป

```bash
npm install
npm run dev
```

เปิดเบราว์เซอร์ไปที่ `http://localhost:5173`

## สร้างไฟล์สำหรับ deploy

```bash
npm run build
```

ไฟล์ที่ build แล้วจะอยู่ในโฟลเดอร์ `dist/`

## วิธีอัปขึ้น GitHub แล้ว deploy ด้วย Vercel

### 1. สร้าง repository บน GitHub

1. ไปที่ [github.com/new](https://github.com/new)
2. ตั้งชื่อ repo เช่น `paobear-app` แล้วกด **Create repository** (ไม่ต้องติ๊ก "Add README" เพราะมีอยู่แล้ว)

### 2. อัปโค้ดขึ้น GitHub

เปิด terminal ในโฟลเดอร์โปรเจกต์นี้ แล้วรันคำสั่งต่อไปนี้ (แทน `YOUR_USERNAME` ด้วยชื่อ GitHub ของคุณ):

```bash
git init
git add .
git commit -m "เริ่มต้นแอปเปาเบียร์"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/paobear-app.git
git push -u origin main
```

### 3. Deploy ด้วย Vercel

1. ไปที่ [vercel.com](https://vercel.com) แล้วล็อกอินด้วยบัญชี GitHub
2. กด **Add New... → Project**
3. เลือก repository `paobear-app` ที่อัปไว้
4. Vercel จะตรวจพบว่าเป็นโปรเจกต์ Vite อัตโนมัติ (Build Command: `npm run build`, Output Directory: `dist`) ไม่ต้องแก้อะไร
5. กด **Deploy** รอประมาณ 1 นาที ก็จะได้ลิงก์แอปพร้อมใช้งานทันที

หลังจากนั้นทุกครั้งที่ `git push` โค้ดใหม่ขึ้น GitHub, Vercel จะ build และ deploy เวอร์ชันใหม่ให้อัตโนมัติ

## โครงสร้างโปรเจกต์

```
src/
  components/   ส่วนประกอบ UI (ฟอร์ม, รายการ, กราฟ, มาสคอตแมว)
  hooks/         useEntries.js — จัดการข้อมูลรายรับ-รายจ่ายผ่าน localStorage
  lib/           helpers.js — หมวดหมู่ ฟังก์ชันจัดรูปแบบวันที่/เงิน
  App.jsx        หน้าหลัก ต่อรวมทุกส่วน
  App.css        สไตล์หลักของแอป
```

## หมายเหตุเรื่องข้อมูล

ข้อมูลรายรับ-รายจ่ายถูกเก็บไว้ใน `localStorage` ของเบราว์เซอร์ที่ใช้งานเท่านั้น หากเปลี่ยนเครื่องหรือเบราว์เซอร์จะไม่เห็นข้อมูลเดิม และถ้าล้างข้อมูลเบราว์เซอร์ (clear browsing data) ข้อมูลในแอปจะหายไปด้วย ถ้าต้องการให้ข้อมูลซิงค์ข้ามอุปกรณ์หรือแชร์กันหลายคน จะต้องเพิ่มฐานข้อมูลออนไลน์ (เช่น Supabase หรือ Firebase) เข้ามาเสริม — บอกได้ถ้าต้องการให้ช่วยทำส่วนนี้เพิ่ม
